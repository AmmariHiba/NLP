# Cross‑Lingual Word Embeddings & Language Identification

## Overview

This project builds English–French semantic representations and evaluates their alignment for cross‑lingual similarity and language identification tasks.

Pipeline:

1. Dataset preparation & cleaning
2. Embedding training (BoW, TF‑IDF, Word2Vec, FastText, GloVe)
3. Cross‑lingual alignment (Orthogonal Procrustes)
4. Semantic analysis & evaluation
5. Language identification

## Requirements

- Python 3.10+
- gensim
- scikit‑learn
- spaCy
- numpy
- matplotlib

## Running the Project

1. Install dependencies
2. Download dataset from [https://tatoeba.org/en/downloads](https://tatoeba.org/en/downloads) selecting english-french 
3. Add it to your folder if wroking with Google collab, and change path if needed 
4. Run notebook `IndividualAssignment.ipynb` cell by cell
